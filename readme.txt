== Installation ==
1. Install typeahead either via the WordPress.org plugin directory, or by uploading the files to your server
2. Enable plugin
3. Try to type some words in your site's search box
4. That's it.You're ready to go!


* 上传插件压缩包到 /wp-content/plugins/ 文件夹并解压。
* 在WordPress后台启用插件或直接在WordPress插件后台搜索Typeahead直接在线安装。
* 在您的网站搜索框随便输入试一试，一个美丽搜索下拉框应该会出现在您面前。
* 您还可以去插件设置页面按您的需要设置插件。
